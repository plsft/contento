# Tailwind CSS v4 Utility Reference (for AI-generated layouts)

## Grid
- `grid`, `grid-cols-1` through `grid-cols-12`
- `col-span-1` through `col-span-12`, `col-start-*`, `col-end-*`
- `gap-1` through `gap-16`, `gap-x-*`, `gap-y-*`

## Flexbox
- `flex`, `flex-col`, `flex-row`, `flex-wrap`
- `items-start`, `items-center`, `items-end`, `items-stretch`
- `justify-start`, `justify-center`, `justify-end`, `justify-between`, `justify-around`
- `grow`, `shrink`, `basis-*`

## Spacing
- Padding: `p-0` through `p-16`, `px-*`, `py-*`, `pt-*`, `pr-*`, `pb-*`, `pl-*`
- Margin: `m-0` through `m-16`, `mx-auto`, `mx-*`, `my-*`, `mt-*`, `mr-*`, `mb-*`, `ml-*`
- Space between: `space-x-*`, `space-y-*`

## Typography
- Size: `text-xs`, `text-sm`, `text-base`, `text-lg`, `text-xl`, `text-2xl`, `text-3xl`, `text-4xl`, `text-5xl`, `text-6xl`
- Weight: `font-light`, `font-normal`, `font-medium`, `font-semibold`, `font-bold`
- Style: `italic`, `not-italic`
- Leading: `leading-none`, `leading-tight`, `leading-snug`, `leading-normal`, `leading-relaxed`, `leading-loose`
- Tracking: `tracking-tighter`, `tracking-tight`, `tracking-normal`, `tracking-wide`, `tracking-wider`
- Alignment: `text-left`, `text-center`, `text-right`, `text-justify`

## Max Width
- `max-w-xs`, `max-w-sm`, `max-w-md`, `max-w-lg`, `max-w-xl`, `max-w-2xl`, `max-w-3xl`, `max-w-4xl`, `max-w-5xl`, `max-w-6xl`, `max-w-7xl`
- `max-w-prose`, `max-w-screen-sm`, `max-w-screen-md`, `max-w-screen-lg`, `max-w-screen-xl`
- `max-w-full`, `max-w-none`

## Width & Height
- `w-full`, `w-screen`, `w-auto`, `w-1/2`, `w-1/3`, `w-2/3`, `w-1/4`, `w-3/4`
- `h-full`, `h-screen`, `h-auto`, `min-h-screen`, `min-h-full`

## Colors (use with text-*, bg-*, border-*)
- Slate: `50`, `100`, `200`, `300`, `400`, `500`, `600`, `700`, `800`, `900`, `950`
- Gray, Stone, Neutral, Zinc — same scale
- Amber, Green, Blue, Indigo, Purple, Red, Orange, Yellow — same scale
- `white`, `black`, `transparent`

## Borders
- `border`, `border-0`, `border-2`, `border-4`
- `border-t`, `border-r`, `border-b`, `border-l`
- `rounded`, `rounded-sm`, `rounded-md`, `rounded-lg`, `rounded-xl`, `rounded-2xl`, `rounded-full`, `rounded-none`

## Shadows
- `shadow-sm`, `shadow`, `shadow-md`, `shadow-lg`, `shadow-xl`, `shadow-2xl`, `shadow-none`

## Display & Visibility
- `block`, `inline-block`, `inline`, `flex`, `grid`, `hidden`
- `overflow-hidden`, `overflow-auto`, `overflow-scroll`

## Position
- `relative`, `absolute`, `fixed`, `sticky`
- `inset-0`, `top-0`, `right-0`, `bottom-0`, `left-0`
- `z-10`, `z-20`, `z-30`, `z-40`, `z-50`

## Responsive Prefixes
- `sm:` — 640px+
- `md:` — 768px+
- `lg:` — 1024px+
- `xl:` — 1280px+
- `2xl:` — 1536px+
