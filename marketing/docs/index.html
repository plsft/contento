Quick Start

Get Contento pSEO generating pages in under five minutes.

What is Contento pSEO?

Contento pSEO is a cloud-hosted programmatic SEO engine that generates thousands of high-quality, niche-targeted pages using AI and structured content schemas. Unlike a traditional CMS where you write each page by hand, Contento automates the entire pipeline: you define a content schema (e.g., "10 Best Tools for {subtopic}"), select your target niches, and the engine cross-multiplies niches with subtopics to produce unique, validated pages at scale.

Contento is not a CMS. It is a purpose-built tool for programmatic SEO. Every generated page includes structured JSON-LD, deterministic title patterns for SERP targeting, custom chrome (your branding header/footer), and a feedback loop from Google Search Console analytics. The entire platform is driven through the contento CLI, with a --json flag on every command for AI agent integration.

Prerequisites

Dependency Version Notes
Node.js 18 or later Required for the CLI
npm 9+ Included with Node.js

Install the CLI

# Install the Contento CLI globally
npm install -g contento

# Verify the installation
contento --version
contento v1.0.0

Log In

# Interactive login (opens browser)
contento login

# Or non-interactive login with API key
contento login --api-key ck_live_your_key_here

Authenticated as george@example.com
API key saved to ~/.contento/config.json
Tip: For CI/CD and AI agent environments, set the CONTENTO_API_KEY environment variable instead of using contento login. The CLI will automatically use it.

First Project Setup

# Create a project
contento projects create \
    --name "DevTools Resources" \
    --domain "example.com" \
    --subdomain "resources.example.com"

  Project created: proj_8f3a...
  Status: pending_dns

  Add this DNS record:
    Type:   CNAME
    Name:   resources
    Target: pseo.contentocms.com

# Browse niches
contento niches list --category "Technology"

# Generate pages
contento collections generate \
    --project proj_8f3a... \
    --schema "idea-list" \
    --niches "niche_01...,niche_02..."

# Publish
contento publish --collection col_9d2f... --mode immediate

The generation pipeline expands subtopics, deduplicates, queues AI generation jobs, validates the output, and publishes. You can monitor real-time progress directly in the terminal.

Configuration

The CLI stores configuration in ~/.contento/config.json:

{
  "apiKey": "ck_live_...",
  "baseUrl": "https://app.contentocms.com"
}

You can also set configuration via environment variables:

Variable Description
CONTENTO_API_KEY API key for authentication (overrides config file)
CONTENTO_API_URL Base URL override (default: https://app.contentocms.com)
