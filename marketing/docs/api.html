Docs REST API (Superseded)

REST API Superseded by CLI

The REST API is now an internal implementation detail. Use the Contento CLI instead.

Looking for the CLI? The CLI Reference is the primary interface for Contento. Install it with npm install -g contento.

Why the Change?

Contento is now a cloud-hosted platform with a CLI-first interface. The CLI handles authentication, request formatting, error handling, and progress streaming automatically. There is no need to construct HTTP requests manually.

For AI Agents

AI agents (Claude Code, Codex, etc.) should use the CLI with the --json flag for machine-readable output. See the Agent Integration guide.

# Non-interactive authentication
$ export CONTENTO_API_KEY=ck_live_abc123...

# All commands support --json for structured output
$ contento projects list --json
$ contento analytics summary --project proj_8f3a... --json
