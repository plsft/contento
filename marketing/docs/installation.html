Install CLI

Install the contento CLI and connect to your account in under a minute.

Prerequisites

DependencyVersionNotes
Node.js18+LTS recommended
npm9+Included with Node.js

Install

# Install globally via npm
npm install -g contento

# Verify installation
contento --version
contento v1.0.0

# Or run without installing (one-shot)
npx contento --version

Authenticate

Interactive login (recommended for humans)

contento login

Opening browser for authentication...
Authenticated as george@example.com
API key saved to ~/.contento/config.json

Non-interactive login (for CI/CD and agents)

# Login with an API key directly
contento login --api-key ck_live_your_key_here

# Or set the environment variable
export CONTENTO_API_KEY=ck_live_your_key_here

# Verify authentication
contento whoami
george@example.com (Growth plan)
Tip: AI agents (Claude Code, Codex, OpenClaw) should use the CONTENTO_API_KEY environment variable. No interactive prompts are needed when this is set.

Configuration

The CLI stores settings in ~/.contento/config.json:

{
  "apiKey": "ck_live_...",
  "baseUrl": "https://app.contentocms.com"
}

Environment Variables

VariableDescriptionDefault
CONTENTO_API_KEY API key (overrides config file) From config
CONTENTO_API_URL Base URL for the Contento API https://app.contentocms.com
NO_COLOR Disable colored output Unset

Global Flags

FlagDescription
--jsonOutput machine-readable JSON instead of formatted tables
--api-url <url>Override the API base URL
--no-colorDisable colored output
-v, --verboseEnable verbose logging
-V, --versionShow CLI version
-h, --helpShow help for any command

Verify Everything Works

# Check authentication
contento whoami

# List your projects
contento projects list

# Browse available niches
contento niches list --category "Technology"

# List available schemas
contento schemas list
