2. The Contento CLI

Contento is an open-source pSEO command-line tool. Its core capabilities include:

  • AI-powered content generation using schema-validated JSON structures and niche-specific context, executed on your machine via your own AI provider key (BYOK).
  • A command-line interface for managing projects, niches, schemas, collections, generation, publishing, and build steps locally.
  • Local static HTML rendering with on-domain output paths for your own site.
  • Optional Google Search Console integration via local OAuth, with analytics cached locally.
  • Progressive publishing with batched rollout, quality gates, and internal linking, all performed locally.
  • Agent-friendly --json output and built-in MCP support.
