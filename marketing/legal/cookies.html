Cookie Policy

Last updated: April 2026

Scope

This policy applies to the Contento marketing website at contento.sh. The Contento CLI runs on your machine and does not set cookies of any kind — nothing in this policy applies to the CLI itself.

1. What We Use

The website is intentionally minimal. We use only:

  • Strictly necessary cookies — if any are required for basic site function (e.g., remembering theme or dismissing a banner). These are first-party and short-lived.
  • localStorage — used by some pages (e.g., the niches browser) to remember UI state in your browser. This data is never sent to our servers.

We do not run advertising cookies, cross-site trackers, social-network pixels, or third-party marketing tags.

2. Analytics

If we run analytics on the marketing site, we use a privacy-respecting, cookieless tool that records aggregated metrics (page views, country-level geo, referrers) without setting persistent identifiers in your browser. We do not link these metrics to any individual user.

3. Server Logs

Like most web infrastructure, our hosting provider records standard request logs (IP, user agent, referrer, timestamp) for security and debugging. Logs are retained for up to 30 days. These are not cookies but are described here for completeness.

4. Controlling Browser Storage

Modern browsers let you block cookies and clear localStorage at any time. The Contento website is fully usable with cookies disabled — you may lose minor UI conveniences (remembered preferences) but no functionality is gated behind cookies.

  • Chrome: Settings → Privacy and security → Cookies and other site data
  • Firefox: Settings → Privacy & Security → Cookies and Site Data
  • Safari: Settings → Privacy → Manage Website Data

5. Changes

We will update this page if we ever start using additional storage technologies. The "Last updated" date at the top reflects the latest revision.
