Headless Mode

Use it your way

Contento works as a traditional CMS with server-rendered pages, or as a headless API powering any frontend. Next.js, Astro, 11ty, mobile apps — anything that speaks HTTP.

  • Content Delivery API — no auth needed for public content
  • Bearer token authentication for management endpoints
  • Posts returned with rendered HTML + Markdown + table of contents
  • Search, comments, media, themes — all accessible via API
  • MCP-compatible — AI agents can create and publish content autonomously
