How it works

Three steps from zero to thousands of indexed pages.

01

Install the CLI

Run npm install -g contento and authenticate with your API key. Create a project, add a CNAME record, and your subdomain goes live in minutes.

02

Choose niches + schemas

Pick from 150+ niche taxonomies with rich context (audience, pain points, subtopics). Select content types: idea lists, checklists, how-to guides, comparisons, FAQs.

03

Generate + publish

AI fills your JSON schemas (not blank pages). 10 concurrent workers, schema validation, deterministic titles. Progressive publishing with batch control.

Built for AI Agents

Claude Code, Codex, OpenClaw, and Co-Work can automate the entire pSEO workflow through the CLI.

agent-workflow.sh 
# An AI agent can run the full workflow non-interactively:

# 1. List available niches (JSON output for parsing)
$ contento niches list --category "Technology" --json

# 2. Create a collection
$ contento collections create \
    --project proj_8f3a \
    --schema idea-list \
    --niches niche_01,niche_02,niche_03 \
    --json

# 3. Generate pages (streams NDJSON progress)
$ contento collections generate coll_a1b2c3 --json
{"completed":12,"total":48,"status":"generating"}
{"completed":48,"total":48,"status":"done","passed":46}

# 4. Publish
$ contento publish coll_a1b2c3 --mode immediate --json
{"published":48,"crossLinks":142,"sitemap":48}

# 5. Check analytics
$ contento analytics top-pages --project proj_8f3a --days 30 --json
