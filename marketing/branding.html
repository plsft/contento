Brand Guidelines

How to represent Contento pSEO consistently — logo, colours, type, voice, and visual language.

Logo

The Contento logo is the word contento set in JetBrains Mono lowercase, followed by a green dot indicator (). The dot represents an active, running system. There is no image file — the logo is always rendered as text.

Usage rules

Do

  • + Always render in JetBrains Mono, lowercase
  • + Always include the green dot indicator
  • + Use #1a1b26 on light or white on dark
  • + Maintain clear space equal to the dot width on all sides

Don't

  • - Use a different typeface or capitalise any letter
  • - Remove or change the colour of the dot
  • - Add shadows, gradients, or effects
  • - Place on busy backgrounds without a solid overlay

Voice & tone

Technical but accessible

We write for developers and SEO practitioners. Use precise terminology, but define it on first use. Never assume the reader knows what a "content schema" is until we tell them.

No marketing fluff

Say what the thing does. "Generates 10,000 pages from a schema" beats "Unlock the power of scalable content." If a sentence could appear in a press release, rewrite it.

Monospace for technical terms

Schema names, CLI commands, file paths, counts, and config values are always set in JetBrains Mono. This visually separates code from prose and reduces ambiguity.

pSEO naming convention

Write "programmatic SEO" on first mention per page, then pSEO thereafter. Numbers are always rendered in monospace — e.g., 150 niches, 10 schemas.

Writing samples

Good

"Contento generates 1 page per location×service combination. A plumber in 12 cities with 8 services produces 96 pages."

Bad

"Supercharge your online presence with our revolutionary AI-powered content generation platform that scales to meet your needs."

Visual language

Grid backgrounds

Use dot-bg and grid-bg classes for subtle texture behind hero sections and feature blocks. Never use photographic backgrounds.

Dashed dividers

Section dividers are 1px dashed, never solid. Matches the technical, blueprint aesthetic.

Console blocks

$ contento generate

generating 2,400 pages...

done in 14.2s

Use the console class for terminal/code output. Dark background, monospace, coloured prompts.

Flat design

Primary Secondary

No box shadows. No gradients. No rounded-xl. Use rounded-lg (8px) max. Buttons are flat with 1px borders.

Card borders

Cards use 1px solid #e5e7eb borders. No elevation, no shadows.

Content cards are defined by their border, not depth. Keep it flat.

Tags & badges

niche schema active cli

Use tag with tag-green or tag-blue. Monospace font, 1px coloured border, tinted background.
